# First Run

When you launch Nous for the first time, you'll see a guided flow that helps you get started.

## First-Run Flow

1. **Welcome** — Introduces Nous and what just happened during installation
2. **Configuration Review** — Shows model assignments and Cortex tier with sensible defaults
3. **First Message** — Prompts you to say something to Nous
4. **Confirmation** — Confirms everything works

The flow completes when you send your first message and receive a response. After that, you're redirected to the main chat interface.

## Skipping First Run

If you already have projects or have completed the first-run flow before, Nous redirects you directly to the chat page.

## What Gets Configured

During first run, the installer generates:

- **Default config** — Cortex tier 3, Ollama as the reasoner provider
- **Storage** — SQLite database in your data directory
- **Projects** — Empty; create your first project from the sidebar

You can adjust model assignments and Cortex tier later in the [Configuration](/docs/guides/config) panel.
