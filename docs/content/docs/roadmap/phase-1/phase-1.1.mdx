# Phase 1.1 — Shared Contracts

**Branch:** `phase-1.1/shared-types-interfaces-config`

**Size:** Medium

**Contributes to:** Milestone 2 (The Structure Exists)

**Prerequisite:** Technology stack decision resolved and documented in architecture reference docs.

---

## Goals

Define and implement the foundational contracts that every subsequent sub-phase builds against. When 1.1 is done, the `self/` structure exists, every cross-layer interface is defined, and the monorepo builds and passes lint.

---

## Scope

### In Scope

- `self/shared/types/` — all core type definitions:
  - PfcTier (0–5)
  - ModelRole (orchestrator, reasoner, tool-advisor, summarizer, embedder, reranker, vision)
  - ProjectType (protocol, intent, hybrid)
  - GovernanceLevel (must, should, may)
  - MemoryType (fact, preference, experience-record, distilled-pattern, task-state)
  - MemoryScope (global, project)
  - Sentiment (strong-positive, weak-positive, neutral, weak-negative, strong-negative)
  - MemoryAccessPolicy (can_read_from, can_be_read_by, inherits_global)
  - NodeType (model-call, tool-execution, quality-gate, human-decision, condition, transform)
  - ExecutionModel (synchronous, streaming, async-batch, scheduled)
  - ProviderType (text, image, video, vision, embedding, external-api)
  - EscalationChannel (in-app, push, signal, slack, sms, email, voice)
  - PackageType (skill, project)

- `self/shared/interfaces/` — cross-layer interface contracts:
  - Cortex: IPfcEngine, ICoreExecutor
  - Memory: IStmStore, ILtmStore, IDistillationEngine, IRetrievalEngine, IKnowledgeIndex, IAccessPolicy
  - Subcortex: IModelRouter, IModelProvider, IToolExecutor, IWorkflowEngine, IProjectStore, IArtifactStore, IScheduler, IEscalationService, ISandbox, IProjectApi
  - Autonomic: IVectorStore, IDocumentStore, IGraphStore, IEmbedder, IRuntime, IConfig, IHealthMonitor

- `self/shared/events/` — base event schemas:
  - SystemEvent, MemoryEvent, ModelEvent, ToolEvent, PfcEvent, ProjectEvent

- `self/shared/errors/` — standardized error types:
  - NousError, PfcDeniedError, MemoryAccessDeniedError, ProviderError, ToolExecutionError, ConfigError

- `self/autonomic/config/` — configuration schema:
  - PFC tier presets (0–5)
  - Model role assignment schema
  - Profile definitions (local-only, remote-only, hybrid)
  - Project configuration defaults
  - Validation and defaults

- Monorepo scaffolding:
  - Package manager configuration
  - Build tooling
  - Linting and formatting
  - Test framework setup
  - CI pipeline (lint, typecheck, test on macOS/Linux/Windows)

### Out of Scope

- Any runtime implementations (those start in 1.2)
- Working functionality (this sub-phase produces contracts and scaffolding, not behavior)
- Web UI, CLI, or any user-facing surface

---

## Definition of Done

- [ ] `self/` folder structure exists with cortex/, memory/, subcortex/, autonomic/, shared/, apps/
- [ ] All shared types defined and documented
- [ ] All cross-layer interfaces defined (stub/empty implementations acceptable)
- [ ] Event schemas defined
- [ ] Error types defined
- [ ] Configuration schema defined with validation
- [ ] Monorepo builds without errors
- [ ] Lint and typecheck pass
- [ ] Test framework configured and at least one example test runs
- [ ] CI pipeline runs on macOS, Linux, and Windows
- [ ] Unit tests for configuration validation
