### Hierarchy Summary

The required hierarchy is: outer shell root -> outer topbar plus left icon rail -> inset app frame -> asset sidebar plus context/workspace canvas plus workspace updates rail -> absolute `Cortex:Principal` drawer overlay above the app frame. This hierarchy is the topology acceptance target; an app-native grid with unrelated rail/sidebar/content/observe/chat regions must either be reconciled to these rows or recorded as an accepted deviation before implementation.
